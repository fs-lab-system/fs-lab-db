# Naming Conventions

## Projects

Format:
fs-lab-<purpose>

Examples:

- fs-lab-bench-data
- fs-lab-test
- fs-lab-core-data

## Tables

- snake_case
- plural nouns

Examples:

- benchmark_runs
- service_metrics

## Columns

- snake_case
- descriptive, no abbreviations

Examples:

- response_time_ms
- request_type
