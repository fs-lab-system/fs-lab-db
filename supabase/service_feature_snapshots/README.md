### service_feature_snapshots

Stores aggregated and engineered service metrics generated by the ML pipeline.

Each row represents a snapshot of service performance at a specific pipeline run.

Fields include:

- latency percentiles (p50, p95, p99)
- tail ratios
- success rate
- run timestamp
- creation timestamp

## Purpose

This schema supports:

- historical performance tracking
- trend analysis
- frontend visualization
- future ML experimentation
